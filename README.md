# Object-tracking-colors
Detects object based on its color using hsv
## Project description
Object tracking based on its color by using hsv(to detect correct color of the object),opencv
## Key features
*Tracking objets by  its color
*Usage of hsv to find accurate color of the object
*It shows the object detected by showing circle around it
*Prints "stop" if object is too close,"front" if object is too far,"left" if object is left,"right" if object is right
## Technologies used
*opencv
*hsv
*imutils
## Run the program
*Get accurate color points(min,max hsv value)
*use this value in program 
*run the program project 3(object tracking based on colors).py
